#================================= Personal Finance Management Web App (Frontend)=====================================#

#  Project Overview:

This is a frontend-only Personal Finance Management web application developed as part of an internship task.  
The application allows users to enter their financial details and view a financial summary and action plan.

This project focuses only on UI development and does not include any backend or database functionality.

---

# Technologies Used:

- HTML5
- CSS3
- JavaScript (Basic – UI interactions only)

---

# Pages Included:

1. Landing Page (index.html) 
   Introduces the application and allows users to start the financial assessment.

2. Financial Assessment Form (onboarding.html)  
   A multi-step form that collects user financial details such as income, expenses, loans, savings, and goals.

3. Dashboard Page (dashboard.html)
   Displays a financial summary including income, expenses, savings rate, debt status, and health score using dummy data.

4. Action Plan Page (plan.html)  
   Shows recommended financial actions categorized into immediate, short-term, and long-term plans.

5. Profile Page (profile.html)  
   Displays basic user profile information (UI only).

---
